# TP3 – Librairie (Frontend Vue)

Ce projet est le **frontend** du TP3 développé avec **Vue 3 et Vite**.
Il consomme une **API REST hébergée sur Vercel**.

---

## Prérequis

- Node.js (version récente recommandée)
- npm

---

## Installation

À la racine du projet :

```bash
npm install

```

## Configuration des variables d’environnement

Créer un fichier `.env` à partir du fichier fourni :


```bash
cp .env.example .env
```

## Lancement en développement

```bash
npm run dev
```
